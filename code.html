                    function getValue(){
                        //make alert invisible
                        document.getElementById("alert").classList.add("invisible");

                    //check if palindrome
                    function checkForPalidrome(userString){
                        userString = userString.toLowerCase();
                        //remove spaces and special chars
                        let regex = /[^a-z0-9]/gi;
                        userString = userString.replace(regex,"");
                    
                        let revString = [];
                        let returnObj = {};
                        for(let i = userString.length - 1; i >= 0; i--){
                            revString+= userString[i];
                        }
                    
                        if(userString == revString){
                            returnObj.msg="Well done! You entered a Palidrome!";
                        }
                        else{
                            returnObj.msg = "Sorry! You did not enter a Palidrome!";
                        }
                        returnObj.reversed = revString;
                        return returnObj;
                    }
                    //display a messasge to the screen
                    function displayMsg(returnObj){
                        document.getElementById("alertHeader").innerHTML = returnObj.msg;
                        document.getElementById("msg").innerHTML = `Your phrase reversed is ${returnObj.reversed}`;
                        //remove invisible class from alert
                        document.getElementById("alert").classList.remove("invisible");
                    }

The Code is structured in three functions.

getValue

Get element from page using document.getElementById and add invisible class to element if class is not currently added. Get user input from page and store into variable userString. Pass userString into function checkForPalidrome() and store the returning object into variable resultObj. Pass variable resultObj into function displayMsg() which will display message to screen.

checkForPalidrome

Start with converting the variable userString to all lowercase characters and reassign result to userString. Use regular expression /[^a-z0-9]/gi to remove any charactors that is not lowercase a-z or 0-9. Use replace funtion on string to replace any character that does not match our regular expression with a empty parentheses. Create a variable revString that holds an array and create a variable returnObj that will hold an object. A for loop is used to loop backwards through the string and store result into revString variable. If the reversed string is equivalent to the user input set property msg of object returnObj to "Well done! You entered a Palidrome!". If the reversed string is not equivalent to the user input set property msg of object returnObj to "Sorry! You did not enter a Palidrome!". Set property reversed of object returnObj to reversed string value. Return reversed string value.

displayMsg

Get element by id using document.getElementById("alertHeader") and set the innerHTML value to the value of returnObj object property msg. Get element by id using document.getElementById("msg") and set the innerHTML to `Your phrase reversed is ${returnObj.reversed}`. Finally we remove the class invisible from element with id "alert" in order to display the message box.
